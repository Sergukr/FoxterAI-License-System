# FoxterAI License Management System

## Компоненты
- **FoxterAI/** - MT4 торговый робот с модулями
- **FoxterAI_Desktop/** - Python приложение администратора
- **FoxterAI_Server/** - Node.js сервер лицензий

## Структура проекта
- Робот → отправляет запросы на сервер для проверки лицензии
- Сервер → хранит и управляет лицензиями в SQLite
- Десктоп → административная панель для управления
